# GitHub Users Application using NextJS

## Setup

-The first thing to do is to clone the repository:

```sh
$ git clone https://github.com/BeyzaaKoroglu/NextJS-GithubUsers.git
```

-Then install the dependencies:

```sh
$ cd NextJS-GithubUsers
$ npm install
```

## Run

```sh
$ npm run dev
```

## Dependencies

- Axios
- Material UI
